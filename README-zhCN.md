Arvin是俺的英文名，作为一个抱大腿的调包开发者，终于在鱼都摸烂了后用Vue.js 3和[Tailwind CSS](https://tailwindcss.com/)建了此个人网站。网站部署在[Vercel](https://vercel.com/home)上。网站的设计遵循以下主要原则。

- 响应式布局。
- 统一设计语言。
- 支持深色模式。
- 国际化: English / 简体中文。
- 无障碍：支持减少动画。
